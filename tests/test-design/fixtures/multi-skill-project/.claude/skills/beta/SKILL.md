# Beta

Summarizes long-form text into concise bullet points.

## Invocation

- **Slash command:** `/beta`
- **Natural language:** "summarize this", "give me the key points"

## Behavior

1. Read the provided text.
2. Extract key points.
3. Present as a bullet list, maximum 10 items.
